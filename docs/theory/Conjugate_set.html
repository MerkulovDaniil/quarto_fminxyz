Conjugate set

Conjugate (dual) set

–ü—É—Å—Ç—å S \subseteq \mathbb{R}^n - –ø—Ä–æ–∏–∑–≤–æ–ª—å–Ω–æ–µ –Ω–µ–ø—É—Å—Ç–æ–µ –º–Ω–æ–∂–µ—Å—Ç–≤–æ. –¢–æ–≥–¥–∞ c–æ–ø—Ä—è–∂–µ–Ω–Ω–æ–µ –∫ –Ω–µ–º—É –º–Ω–æ–∂–µ—Å—Ç–≤–æ –æ–ø—Ä–µ–¥–µ–ª—è–µ—Ç—Å—è, –∫–∞–∫:

S^* = \{y \ \in \mathbb{R}^n \mid \langle y, x\rangle \ge -1 \;\; \forall x \in S\}

Double conjugate set

–ú–Ω–æ–∂–µ—Å—Ç–≤–æ S^{**} –Ω–∞–∑—ã–≤–∞–µ—Ç—Å—è –≤—Ç–æ—Ä—ã–º —Å–æ–ø—Ä—è–∂–µ–Ω–Ω—ã–º –∫ –º–Ω–æ–∂–µ—Å—Ç–≤—É S, –µ—Å–ª–∏:

S^{**} = \{y \ \in \mathbb{R}^n \mid \langle y, x\rangle \ge -1 \;\; \forall x \in S^*\}

Inter-conjugate and self-conjugate sets

  • –ú–Ω–æ–∂–µ—Å—Ç–≤–∞ S_1 –∏ S_2 –Ω–∞–∑—ã–≤–∞—é—Ç—Å—è –≤–∑–∞–∏–º–æ—Å–æ–ø—Ä—è–∂–µ–Ω–Ω—ã–º–∏, –µ—Å–ª–∏ S_1^* = S_2, S_2^* = S_1.
  • –ú–Ω–æ–∂–µ—Å—Ç–≤–æ S –Ω–∞–∑—ã–≤–∞–µ—Ç—Å—è —Å–∞–º–æ—Å–æ–ø—Ä—è–∂–µ–Ω–Ω—ã–º, –µ—Å–ª–∏ S^{*} = S.

Properties

  • –°–æ–ø—Ä—è–∂–µ–Ω–Ω–æ–µ –º–Ω–æ–∂–µ—Å—Ç–≤–æ –≤—Å–µ–≥–¥–∞ –∑–∞–º–∫–Ω—É—Ç–æ, –≤—ã–ø—É–∫–ª–æ –∏ —Å–æ–¥–µ—Ä–∂–∏—Ç –Ω—É–ª—å.

  • –î–ª—è –ø—Ä–æ–∏–∑–≤–æ–ª—å–Ω–æ–≥–æ –º–Ω–æ–∂–µ—Å—Ç–≤–∞ S \subseteq \mathbb{R}^n:

    S^{**} = \overline{ \mathbf{conv} (S \cup \{0\}) }

  • –ï—Å–ª–∏ S_1 \subseteq S_2, —Ç–æ S_2^* \subseteq S_1^*.

  • \left( \bigcup\limits_{i=1}^m S_i \right)^* = \bigcap\limits_{i=1}^m S_i^*

  • –ï—Å–ª–∏ S - –∑–∞–º–∫–Ω—É—Ç–æ, –≤—ã–ø—É–∫–ª–æ, –≤–∫–ª—é—á–∞–µ—Ç 0, —Ç–æ S^{**} = S.

  • S^* = \left(\overline{S}\right)^*

Examples

1

–î–æ–∫–∞–∑–∞—Ç—å, —á—Ç–æ S^* = \left(\overline{S}\right)^*.

–†–µ—à–µ–Ω–∏–µ:

  • S \subset \overline{S} \rightarrow \left(\overline{S}\right)^* \subset S^*
  • –ü—É—Å—Ç—å p \in S^* –∏ x_0 \in \overline{S}, x_0 = \underset{k \to \infty}{\operatorname{lim}} x_k. –¢–æ–≥–¥–∞ –≤ —Å–∏–ª—É –Ω–µ–ø—Ä–µ—Ä—ã–≤–Ω–æ—Å—Ç–∏ —Ñ—É–Ω–∫—Ü–∏–∏ f(x) = p^Tx, –∏–º–µ–µ–º: p^T x_k \ge -1 \to p^Tx_0 \ge -1. –ó–Ω–∞—á–∏—Ç, p \in \left(\overline{S}\right)^*, –æ—Ç—Å—é–¥–∞ S^* \subset \left(\overline{S}\right)^*

2

–î–æ–∫–∞–∑–∞—Ç—å, —á—Ç–æ \left( \mathbf{conv}(S) \right)^* = S^*.

–†–µ—à–µ–Ω–∏–µ:

  • S \subset \mathbf{conv}(S) \to \left( \mathbf{conv}(S) \right)^* \subset S^*

  • –ü—É—Å—Ç—å p \in S^*, x_0 \in \mathbf{conv}(S), —Ç.–µ. x_0 = \sum\limits_{i=1}^k\theta_i x_i \mid x_i \in S, \sum\limits_{i=1}^k\theta_i = 1, \theta_i \ge 0.

    –ó–Ω–∞—á–∏—Ç, p^T x_0 = \sum\limits_{i=1}^k\theta_i p^Tx_i \ge \sum\limits_{i=1}^k\theta_i (-1) = 1 \cdot (-1) = -1. –ó–Ω–∞—á–∏—Ç, p \in \left( \mathbf{conv}(S) \right)^*, –æ—Ç—Å—é–¥–∞ S^* \subset \left( \mathbf{conv}(S) \right)^*

3

–î–æ–∫–∞–∑–∞—Ç—å, —á—Ç–æ –µ—Å–ª–∏ B(0,r) - —à–∞—Ä —Ä–∞–¥–∏—É—Å–∞ r –ø–æ –Ω–µ–∫–æ—Ç–æ—Ä–æ–π –Ω–æ—Ä–º–µ —Å —Ü–µ–Ω—Ç—Ä–æ–º –≤ –Ω—É–ª–µ, —Ç–æ \left( B(0,r) \right)^* = B(0,1/r).

–†–µ—à–µ–Ω–∏–µ:

  • –ü—É—Å—Ç—å B(0,r) = X, B(0,1/r) = Y. –í–æ–∑—å–º–µ–º –≤–µ–∫—Ç–æ—Ä –Ω–æ—Ä–º–∞–ª–∏ p \in X^*, —Ç–æ–≥–¥–∞ –¥–ª—è –ª—é–±–æ–≥–æ x \in X: p^Tx \ge -1.

  • –ò–∑ –≤—Å–µ—Ö —Ç–æ—á–µ–∫ —à–∞—Ä–∞ X –≤–æ–∑—å–º–µ–º —Ç–∞–∫—É—é x \in X, —á—Ç–æ —Å–∫–∞–ª—è—Ä–Ω–æ–µ –ø—Ä–æ–∏–∑–≤–µ–¥–µ–Ω–∏–µ –µ—ë –Ω–∞ p: p^Tx –±—ã–ª–æ –±—ã –º–∏–Ω–∏–º–∞–ª—å–Ω–æ, —Ç–æ–≥–¥–∞ —ç—Ç–æ —Ç–æ—á–∫–∞ x = -\frac{p}{\|p\|}r

    p^T x = p^T \left(-\frac{p}{\|p\|}r \right) = -\|p\|r \ge -1

    \|p\| \le \frac{1}{r} \in Y

    –ó–Ω–∞—á–∏—Ç, X^* \subset Y.

  • –¢–µ–ø–µ—Ä—å –ø—É—Å—Ç—å p \in Y. –ù–∞–º –Ω–∞–¥–æ –ø–æ–∫–∞–∑–∞—Ç—å, —á—Ç–æ p \in X^*, —Ç.–µ. \langle p, x\rangle \geq -1. –î–æ—Å—Ç–∞—Ç–æ—á–Ω–æ –ø—Ä–∏–º–µ–Ω–∏—Ç—å –Ω–µ—Ä–∞–≤–µ–Ω—Å—Ç–≤–æ –ö–æ—à–∏ - –ë—É–Ω—è–∫–æ–≤—Å–∫–æ–≥–æ:

    \|\langle p, x\rangle\| \leq \|p\| \|x\| \leq \dfrac{1}{r} \cdot r = 1

    –ü–æ—Å–ª–µ–¥–Ω–µ–µ –∏—Å—Ö–æ–¥–∏—Ç –∏–∑ —Ç–æ–≥–æ, —á—Ç–æ p \in B(0,1/r), –∞ x \in B(0,r).

    –ó–Ω–∞—á–∏—Ç, Y \subset X^*.

Dual cones

–°–æ–ø—Ä—è–∂–µ–Ω–Ω—ã–º –∫–æ–Ω—É—Å–æ–º –∫ –∫–æ–Ω—É—Å—É K –Ω–∞–∑—ã–≤–∞–µ—Ç—Å—è —Ç–∞–∫–æ–µ –º–Ω–æ–∂–µ—Å—Ç–≤–æ K^*, —á—Ç–æ:

K^* = \left\{ y \mid \langle x, y\rangle \ge 0 \;\;\;\; \forall x \in K\right\}

–ß—Ç–æ–±—ã –ø–æ–∫–∞–∑–∞—Ç—å, —á—Ç–æ —ç—Ç–æ –æ–ø—Ä–µ–¥–µ–ª–µ–Ω–∏–µ –Ω–µ–ø–æ—Å—Ä–µ–¥—Å—Ç–≤–µ–Ω–Ω–æ —Å–ª–µ–¥—É–µ—Ç –∏–∑ —Ç–µ–æ—Ä–∏–∏ –≤—ã—à–µ, –≤—Å–ø–æ–º–Ω–∏–º, —á—Ç–æ —Ç–∞–∫–æ–µ —Å–æ–ø—Ä—è–∂–µ–Ω–Ω–æ–µ –º–Ω–æ–∂–µ—Å—Ç–≤–æ –∏ —á—Ç–æ —Ç–∞–∫–æ–µ –∫–æ–Ω—É—Å \forall \lambda > 0

\{y \ \in \mathbb{R}^n \mid \langle y, x\rangle \ge -1 \;\; \forall x \in S\} \to \{\lambda y \ \in \mathbb{R}^n \mid \langle y, x\rangle \ge -\dfrac{1}{\lambda} \;\; \forall x \in S\}

Dual cones properties

  • –ü—É—Å—Ç—å K - –∑–∞–º–∫–Ω—É—Ç—ã–π –≤—ã–ø—É–∫–ª—ã–π –∫–æ–Ω—É—Å. –¢–æ–≥–¥–∞ K^{**} = K.

  • –î–ª—è –ø—Ä–æ–∏–∑–≤–æ–ª—å–Ω–æ–≥–æ –º–Ω–æ–∂–µ—Å—Ç–≤–∞ S \subseteq \mathbb{R}^n –∏ –∫–æ–Ω—É—Å–∞ K \subseteq \mathbb{R}^n:

    \left( S + K \right)^* = S^* \cap K^*

  • –ü—É—Å—Ç—å K_1, \ldots, K_m - –∫–æ–Ω—É—Å—ã –≤ \mathbb{R}^n, —Ç–æ–≥–¥–∞:

    \left( \sum\limits_{i=1}^m K_i \right)^* = \bigcap\limits_{i=1}^m K_i^*

  • –ü—É—Å—Ç—å K_1, \ldots, K_m - –∫–æ–Ω—É—Å—ã –≤ \mathbb{R}^n. –ü—É—Å—Ç—å —Ç–∞–∫–∂–µ –∏—Ö –ø–µ—Ä–µ—Å–µ—á–µ–Ω–∏–µ –∏–º–µ–µ—Ç –≤–Ω—É—Ç—Ä–µ–Ω–Ω—é—é —Ç–æ—á–∫—É, —Ç–æ–≥–¥–∞:

    \left( \bigcap\limits_{i=1}^m K_i \right)^* = \sum\limits_{i=1}^m K_i^*

Examples

–ù–∞–π—Ç–∏ —Å–æ–ø—Ä—è–∂–µ–Ω–Ω–π –∫–æ–Ω—É—Å –¥–ª—è –º–æ–Ω–æ—Ç–æ–Ω–Ω–æ–≥–æ –Ω–µ–æ—Ç—Ä–∏—Ü–∞—Ç–µ–ª—å–Ω–æ–≥–æ –∫–æ–Ω—É—Å–∞:

K = \left\{ x \in \mathbb{R}^n \mid x_1 \ge x_2 \ge \ldots \ge x_n \ge 0\right\}

–†–µ—à–µ–Ω–∏–µ:

–ó–∞–º–µ—Ç–∏–º, —á—Ç–æ:

\sum\limits_{i=1}^nx_iy_i = y_1 (x_1-x_2) + (y_1 + y_2)(x_2 - x_3) + \ldots + (y_1 + y_2 + \ldots + y_{n-1})(x_{n-1} - x_n) + (y_1 + \ldots + y_n)x_n

–¢–∞–∫ –∫–∞–∫ –≤ –ø—Ä–µ–¥—Å—Ç–∞–≤–ª–µ–Ω–Ω–æ–π —Å—É–º–º–µ –≤ –∫–∞–∂–¥–æ–º —Å–ª–∞–≥–∞–µ–º–æ–º –≤—Ç–æ—Ä–æ–π –º–Ω–æ–∂–∏—Ç–µ–ª—å –Ω–µ–æ—Ç—Ä–∏—Ü–∞—Ç–µ–ª—å–Ω—ã–π, —Ç–æ:

y_1 \ge 0, \;\; y_1 + y_2 \ge 0, \;\;\ldots, \;\;y_1 + \ldots + y_n \ge 0

–ó–Ω–∞—á–∏—Ç, K^* = \left\{ y \mid \sum\limits_{i=1}^k y_i \ge 0, k = \overline{1,n}\right\}

Polyhedra

–ú–Ω–æ–∂–µ—Å—Ç–≤–æ —Ä–µ—à–µ–Ω–∏–π —Å–∏—Å—Ç–µ–º—ã –ª–∏–Ω–µ–π–Ω—ã—Ö –Ω–µ—Ä–∞–≤–µ–Ω—Å—Ç–≤ –∏ —Ä–∞–≤–µ–Ω—Å—Ç–≤ –ø—Ä–µ–¥—Å—Ç–∞–≤–ª—è–µ—Ç —Å–æ–±–æ–π –º–Ω–æ–≥–æ–≥—Ä–∞–Ω–Ω–∏–∫:

Ax \preceq b, \;\;\; Cx = d

–ó–¥–µ—Å—å A \in \mathbb{R}^{m\times n}, C \in \mathbb{R}^{p \times n} , –∞ –Ω–µ—Ä–∞–≤–µ–Ω—Å—Ç–≤–æ - –ø–æ—ç–ª–µ–º–µ–Ω—Ç–Ω–æ–µ.

–¢–µ–æ—Ä–µ–º–∞:

–ü—É—Å—Ç—å x_1, \ldots, x_m \in \mathbb{R}^n. –°–æ–ø—Ä—è–∂–µ–Ω–Ω—ã–º –∫ –º–Ω–æ–≥–æ–≥—Ä–∞–Ω–Ω–æ–º—É –º–Ω–æ–∂–µ—Å—Ç–≤—É:

S = \mathbf{conv}(x_1, \ldots, x_k) + \mathbf{cone}(x_{k+1}, \ldots, x_m)

—è–≤–ª—è–µ—Ç—Å—è –ø–æ–ª–∏—ç–¥—Ä (–º–Ω–æ–≥–æ–≥—Ä–∞–Ω–Ω–∏–∫):

S^* = \left\{ p \in \mathbb{R}^n \mid \langle p, x_i\rangle \ge -1, i = \overline{1,k} ; \langle p, x_i\rangle \ge 0, i = \overline{k+1,m} \right\}

–î–æ–∫–∞–∑–∞—Ç–µ–ª—å—Å—Ç–≤–æ:

  • –ü—É—Å—Ç—å S = X, S^* = Y. –í–æ–∑—å–º–µ–º –Ω–µ–∫–æ—Ç–æ—Ä—ã–π p \in X^*, —Ç–æ–≥–¥–∞ \langle p, x_i\rangle \ge -1, i = \overline{1,k}. –í —Ç–æ –∂–µ –≤—Ä–µ–º—è –¥–ª—è –ª—é–±—ã—Ö \theta > 0, i = \overline{k+1,m}:

    \langle p, x_i\rangle \ge -1 \to \langle p, \theta x_i\rangle \ge -1

    \langle p, x_i\rangle \ge -\frac{1}{\theta} \to \langle p, x_i\rangle \ge 0

    –ó–Ω–∞—á–∏—Ç, p \in Y \to X^* \subset Y.

  • –ü—É—Å—Ç—å, –Ω–∞–ø—Ä–æ—Ç–∏–≤, p \in Y. –î–ª—è –ª—é–±–æ–π —Ç–æ—á–∫–∏ x \in X:

    x = \sum\limits_{i=1}^m\theta_i x_i \;\;\;\;\;\; \sum\limits_{i=1}^k\theta_i = 1, \theta_i \ge 0

    –ó–Ω–∞—á–∏—Ç:

    \langle p, x\rangle = \sum\limits_{i=1}^m\theta_i \langle p, x_i\rangle = \sum\limits_{i=1}^k\theta_i \langle p, x_i\rangle + \sum\limits_{i=k+1}^m\theta_i \langle p, x_i\rangle \ge \sum\limits_{i=1}^k\theta_i (-1) + \sum\limits_{i=1}^k\theta_i \cdot 0 = -1

    –ó–Ω–∞—á–∏—Ç, p \in X^* \to Y \subset X^*.

5

–ù–∞–π—Ç–∏ –∏ –∏–∑–æ–±—Ä–∞–∑–∏—Ç—å –Ω–∞ –ø–ª–æ—Å–∫–æ—Å—Ç–∏ –º–Ω–æ–∂–µ—Å—Ç–≤–æ, —Å–æ–ø—Ä—è–∂–µ–Ω–Ω–æ–µ –∫ –º–Ω–æ–≥–æ–≥—Ä–∞–Ω–Ω–æ–º—É –∫–æ–Ω—É—Å—É:

S = \mathbf{cone} \left\{ (-3,1), (2,3), (4,5)\right\}

–†–µ—à–µ–Ω–∏–µ:

–ò—Å–ø–æ–ª—å–∑—É—è —Ç–µ–æ—Ä–µ–º—É –≤—ã—à–µ:

S^* = \left\{ -3p_1 + p_2 \ge 0, 2p_1 + 3p_2 \ge 0, 4p_1+5p_2 \ge 0 \right\}

–õ–µ–º–º–∞ (—Ç–µ–æ—Ä–µ–º–∞) –§–∞—Ä–∫–∞—à–∞ (–§–∞—Ä–∫–∞—à–∞ - –ú–∏–Ω–∫–æ–≤—Å–∫–æ–≥–æ)

–ü—É—Å—Ç—å A \in \mathbb{R}^{m\times n}, b \in \mathbb{R}^m. –¢–æ–≥–¥–∞ –∏–º–µ–µ—Ç —Ä–µ—à–µ–Ω–∏–µ –æ–¥–Ω–∞ –∏ —Ç–æ–ª—å–∫–æ –æ–¥–Ω–∞ –∏–∑ —Å–ª–µ–¥—É—é—â–∏—Ö –¥–≤—É—Ö —Å–∏—Å—Ç–µ–º:

1) \; Ax = b, x \ge 0\;\;\;\;\;\;

2) \; p^\top A \ge 0, \langle p,b\rangle < 0

Ax = b –ø—Ä–∏ x \geq 0 –æ–∑–Ω–∞—á–∞–µ—Ç, —á—Ç–æ b –ª–µ–∂–∏—Ç –≤ –∫–æ–Ω—É—Å–µ, –Ω–∞—Ç—è–Ω—É—Ç—ã–º –Ω–∞ —Å—Ç–æ–ª–±—Ü—ã –º–∞—Ç—Ä–∏—Ü—ã A.

pA \geq 0, \; \langle p, b \rangle < 0 –æ–∑–Ω–∞—á–∞–µ—Ç, —á—Ç–æ —Å—É—â–µ—Å—Ç–≤—É–µ—Ç —Ä–∞–∑–¥–µ–ª—è—é—â–∞—è –≥–∏–ø–µ—Ä–ø–ª–æ—Å–∫–æ—Å—Ç—å –º–µ–∂–¥—É –≤–µ–∫—Ç–æ—Ä–æ–º b –∏ –∫–æ–Ω—É—Å–æ–º –∏–∑ —Å—Ç–æ–ª–±—Ü–æ–≤ –º–∞—Ç—Ä–∏—Ü—ã A.

–°–ª–µ–¥—Å—Ç–≤–∏–µ:

–ü—É—Å—Ç—å A \in \mathbb{R}^{m\times n}, b \in \mathbb{R}^m. –¢–æ–≥–¥–∞ –∏–º–µ–µ—Ç —Ä–µ—à–µ–Ω–∏–µ –æ–¥–Ω–∞ –∏ —Ç–æ–ª—å–∫–æ –æ–¥–Ω–∞ –∏–∑ —Å–ª–µ–¥—É—é—â–∏—Ö –¥–≤—É—Ö —Å–∏—Å—Ç–µ–º:

1) \; Ax \le b \;\;\;\;\;\;\;\;\;\;\;\;\;\;\;\;\;\;\;\;\;\;\;\;

2) \; p^\top A = 0, \langle p,b\rangle < 0, p \ge 0

–ï—Å–ª–∏ –≤ –∑–∞–¥–∞—á–µ –ª–∏–Ω–µ–π–Ω–æ–≥–æ –ø—Ä–æ–≥—Ä–∞–º–º–∏—Ä–æ–≤–∞–Ω–∏—è –Ω–∞ –º–∏–Ω–∏–º—É–º –¥–æ–ø—É—Å—Ç–∏–º–æ–µ –º–Ω–æ–∂–µ—Å—Ç–≤–æ –Ω–µ–ø—É—Å—Ç–æ –∏ —Ü–µ–ª–µ–≤–∞—è —Ñ—É–Ω–∫—Ü–∏—è –æ–≥—Ä–∞–Ω–∏—á–µ–Ω–∞ –Ω–∞ –Ω—ë–º —Å–Ω–∏–∑—É, —Ç–æ –∑–∞–¥–∞—á–∞ –∏–º–µ–µ—Ç —Ä–µ—à–µ–Ω–∏–µ.
